# LoRaMix_Box

Boîtier 3D paramétrique pour la carte **LoRa Mix V2 Premium** de Sebydocky.

## Objectif

Créer un boîtier imprimable en 3D :

- clipsable
- entièrement paramétrique
- compatible FreeCAD
- exportable en STL et STEP
- adapté au PCB LoRa Mix

## Fonctionnalités

- Base clipsable
- Couvercle
- Colonnettes PCB
- Découpe SMA
- Découpe RJ45
- Découpes borniers
- Porte-fusibles
- Ventilation
- Gravure LoRa Mix
